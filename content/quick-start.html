Quick Start Guide

A short introduction to Phaser

If you haven't used Phaser before, you should check out the vast amount of tutorials and starting guides that are available. The previous version of Phaser is maintained by the community, and thus titled Phaser Community Edition.

Phaser is a game engine for writing games in JavaScript, and all you have to do to set it up is include the phaser.js file in your game's folder. Then, you can create an index.html file with the line <script src="phaser.js"></script> included. You can then create a game.js file as well, where you will write the code to your game, and include that in the index.html. This is basically all that is needed to get Phaser up and running, continue with the next part to find out how to actually create your game.

Setting up scenes

Examples Check out the Phaser 3 examples directory for this topic.

If you're familiar with Phaser 2, you know how states are used in it to control the flow of the game. Phaser 3 has scenes instead of states and these are even more versatile, and can be created several different ways (from a function, an object, a class, an instance, see the examples/scenes directory). Here, I will use what I believe is the simplest one: creating from an object, where you only have to specify the functions it uses for preload and create.

var config = {
    type: Phaser.WEBGL,
    width: 800,
    height: 600,
    scene: {
        preload: preload,
        create: create
    }
};

var game = new Phaser.Game(config);

function preload () {
	//preload
}

function create () {
    //create
}

As you can see from the code above, the actual game instance is created with new Phaser.Game(), and you have to provide a config object with the settings. Of course you don't have to declare this before, it can be part of the Phaser.Game() statement.

In Phaser 3, things like the GameObjectFactory and Loader belong to the scene instead of the game instance. In preload for example, you load an image with this.load.image().

Loading

Examples Check out the Phaser 3 examples directory for this topic.

You can load files just like in Phaser 2, using this.load.image(key, filename). The key is later used for identification, and the filename will have to include the path of the file if it's not in the same directory as index.html. You can use another method, where you specify where your image files are kept and then only give a list of keys, Phaser will find image files in that folder that have the same name as the keys and will import them.

this.load.path = 'assets/pics/';

this.load.image([
	{ key: 'sukasuka-chtholly' },
	{ key: 'taikodrummaster', extension: 'jpg' }
]);

This way, you don't include the extention of the file, and you have to specify in case it isn't the default .png; if you've got all of your images in the same folder, you may find this very useful.

Spritesheets and Texture atlases also have parameters you have to specify in order to load them, check out the examples directory know more.

function load() {
	this.load.spritesheet('explosion', 'assets/sprites/explosion.png', { frameWidth: 64, frameHeight: 64, endFrame: 23 });
	this.load.atlas('megaset', 'assets/atlas/megaset-0.png', 'assets/atlas/megaset-0.json');
}

Creating objects

Examples Check out the Phaser 3 examples directory for this topic.

Creating sprites is just like in Phaser 2, only the GameObjectFactory is accessed in a different way, but bunny = this.add.sprite(400, 300, 'bunny'); works like you would expect, creating a sprite at coordinates 400, 300 from an image loaded with the key 'bunny'. Obviously you have to load the image you want to use as a texture before you create the sprite (unless you're doing some extra tricky stuff). You can also use this.make.sprite(config) with a custom config object, where you can specify things like x and y, flipping the image, or choosing one image at random from an array.

var config2 = {
	key: [ 'bunny', 'atari', 'logo' ],
	x: 400,
	y: 300,
	flipX: true
};

You can also easily create texts, and set their properties with a config object.

this.add.text(100, 200, 'Phaser', { fontFamily: 'Arial', fontSize: 64, color: '#00ff00' });
